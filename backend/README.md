# Product Data Explorer

Full-stack application built using:

## Backend
- NestJS
- Prisma ORM
- PostgreSQL
- REST APIs (CRUD)
- Swagger API Docs

## Frontend
- React
- Axios
- Product CRUD UI
- Runs on http://localhost:3000

## Features
- Create Product
- View Products
- Update Product
- Delete Product

## How to Run

### Backend
```bash
cd backend
npm install
npx prisma migrate dev
npm run start:dev
